				DOCUMENTO 30.  FRAGMENTO DE CARTA DIRIGIDA A ALEJANDRO DEHOLLAIN, DESDE CHARLESTON, REFERIDA AL VIAJE DE RE­GRESO A AMÉRICA.*

				
            


          

            
            

		
        
			
			[Charleston,1°, ó 10, de enero de 1807].


[Al señor Alejandro Dehollain-Arnoux].


... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ...
Le escribo para participarle mi llegada a este puerto, después de
haber pasado una navegación muy desagradable por el mal tiempo
y mi mala salud, sintiéndome de calenturas ... ... ... ... ... ... ... ... ... ... ... ... ... ... ...
... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ... ...


[BOLIVAR].


*   Archivo del Libertador, vol. 114, f° 66 (cartas en inglés y en francés). Se trata de un fragmento de carta de Bolívar, reproducido en otra autó­grafa de Alejandro Dehollain-Arnoux para el Libertador fechada en Lon­dres a 20 de agosto de 1820. "En su carta, escrita en francés, Dehollain transcribía en español un párrafo de una de las cartas que Bolívar le había enviado desde Charleston, puerto de Carolina del Sur, en los Esta­dos Unidos, el I9 y el 10 de enero de 1807, sin precisar a cuál de las dos pertenecía. El fragmento, tal como lo transcribió Dehollain, decía así exac­tamente: "... por participarme su llegada a este puerto después de haver passado una navegación muy desagradable por el mal tiempo y su mala salud, se sintiendo de calenturas. . ." Como se ve, Dehollain había puesto en tercera persona lo que Bolívar hubo, evidentemente, de escribir en pri­mera. Restablecemos el sentido lógico de sus palabras, tal como fueron publicadas en Fundación John Boulton, Cartas del Libertador (1803-1830), tomo XII, citado, pág. 21. Sobre la identidad del destinatario, véase la nota principal del doc. N9 26.
